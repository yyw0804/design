# yw Brand Guidelines

> 이 문서는 yw의 브랜드 언어를 일관되게 유지하기 위한 기준입니다.  
> 디자인, 카피, 커뮤니케이션 전반에 적용됩니다.

---

## 1. Brand Essence

### Core Idea

**"오래 곁에 두고 싶은 것들만 만든다."**

yw의 모든 결정은 이 문장으로 돌아옵니다.  
제품의 형태, 소재의 선택, 말하는 방식, 보여주는 장면 — 전부 이 기준 안에 있습니다.

### Brand Personality

| 속성 | 설명 |
|---|---|
| **조용한** | 먼저 말을 걸지 않는다. 하지만 한번 눈에 들어오면 떠나지 않는다. |
| **절제된** | 하나를 더하기보다 하나를 덜어내는 방향을 선택한다. |
| **감각적** | 시각보다 촉각에 가깝다. 보기 좋은 것이 아니라 손에 닿았을 때 좋은 것. |
| **희소한** | 많이 만들지 않는다. 그래서 가지고 있는 사람이 특별하다. |

## 2. Logo & Typography

### 로고

- 로고타입은 소문자 `yw`를 기본으로 합니다.
- 여백을 침해하지 않도록 로고 주변 최소 여백은 로고 높이의 **1x** 이상 확보합니다.
- 배경색에 따라 **Black / White** 두 가지 버전만 사용합니다.
- 로고에 그림자, 외곽선, 그라디언트를 적용하지 않습니다.

### 서체

#### 주요 서체 (국문)

| 용도 | 서체 | 웨이트 |
|---|---|---|
| 헤드라인 | Noto Serif KR | Light / Regular |
| 본문 | Noto Sans KR | Light |
| 캡션 / 레이블 | Noto Sans KR | Regular |

#### 주요 서체 (영문)

| 용도 | 서체 | 웨이트 |
|---|---|---|
| 헤드라인 | Cormorant Garamond | Light / Italic |
| 본문 | DM Sans | Light |
| 캡션 / 레이블 | DM Sans | Regular |

### 타이포그래피 원칙

- 자간은 넓게. 행간은 여유있게. 빈 공간을 두려워하지 않습니다.
- 볼드는 강조가 아니라 위계를 위해서만 씁니다.
- 대문자 단독 사용은 레이블·태그에 한정합니다.

## 3. Color Palette

### Primary

| 이름 | HEX | 용도 |
|---|---|---|
| **yw Black** | `#1A1A1A` | 주요 텍스트, 로고 |
| **yw White** | `#F7F5F2` | 배경, 여백 |
| **yw Stone** | `#D6CFC6` | 보조 배경, 구분선 |

### Secondary

| 이름 | HEX | 용도 |
|---|---|---|
| **Ash** | `#8C8680` | 캡션, 서브 텍스트 |
| **Clay** | `#B5967A` | 포인트 컬러 (최소 사용) |
| **Fog** | `#E8E4DF` | 카드 배경, 레이어 |

### 컬러 사용 원칙

- 전체 레이아웃의 **80% 이상은 White + Stone 계열**로 구성합니다.
- Clay는 CTA, 키 메시지 포인트에 한해 절제적으로 사용합니다.
- 컬러를 겹쳐 쓸 때는 채도 차이보다 명도 차이로 위계를 만듭니다.

## 4. Photography & Visual Direction

### 촬영 방향

- **자연광** 기반. 스튜디오 조명을 사용할 경우 그림자가 자연스럽게 드리워지도록 합니다.
- 배경은 **텍스처가 있는 중성색** (린넨, 석재, 무광 콘크리트 계열).
- 제품이 공간 안에 **이미 있었던 것처럼** 연출합니다. 과하게 스타일링하지 않습니다.
- 클로즈업 비율을 높여 소재와 표면의 감각을 전달합니다.

### 피해야 할 것

- 과채도 보정, 과도한 필터
- 백플랫(White seamless) 단독 제품 컷 위주의 구성
- 인물이 카메라를 바라보는 사진
- 여러 제품을 한 컷에 빽빽하게 채우는 구성

### 비율 가이드

| 채널 | 비율 | 비고 |
|---|---|---|
| Instagram Feed | 4:5 | 기본 |
| Instagram Story | 9:16 | |
| 웹 배너 | 16:9 | |
| 상세페이지 | 1:1 / 3:4 | 혼용 가능 |

## 5. Voice & Tone

### 핵심 원칙

yw는 **설명하지 않습니다**. 장면을 보여줍니다.  
독자에게 결론을 건네지 않고, 감각이 닿을 자리를 남겨둡니다.

### 써야 할 언어

- 감각적이고 구체적인 묘사
- 짧은 문장. 여운이 남는 마무리.
- 관찰에서 시작하는 서술 (현상 → 감각 → 여백)

### 쓰지 않는 언어

- 수사적 질문 ("이런 적 없으신가요?")
- 과도한 형용사 ("감각적인", "세련된", "프리미엄")
- 명시적 감정 결론 ("더 여유로운 하루를 만들어드립니다")
- 유행어, 줄임말, 이모지

### 톤 비교

| ✗ 쓰지 않는 방식 | ✓ yw의 방식 |
|---|---|
| "당신의 공간을 더 특별하게" | "아침에 제일 먼저 손이 가는 컵이 있다." |
| "감성 가득한 홈 데코" | "빛이 닿는 각도에 따라 색이 달라진다." |
| "프리미엄 소재로 만든 테이블웨어" | "무겁지 않은데 가볍지도 않다." |

## 6. Naming Convention

### 제품명

- 제품명은 **영문 소문자** 또는 **고유명사** 형태를 기본으로 합니다.
- 기능 설명은 제품명에 포함하지 않습니다.
- 가능하면 한 단어. 두 단어일 경우 공백 없이 붙여 씁니다.

**예시**  
`vale` / `olen` / `maru` / `stillwater`

### 컬렉션명

- 시즌 컬렉션은 **정서적 상태 또는 장소명**에서 가져옵니다.
- 연도나 시즌 넘버를 제목에 드러내지 않습니다.

**예시**  
`After Rain` / `Still Morning` / `Inland`

## 7. Channel Guidelines

### Instagram

- 피드는 **3열 기준 색조와 여백의 흐름**이 이어지도록 구성합니다.
- 캡션은 2–4문장 이내. 해시태그는 최소화하거나 댓글에 분리합니다.
- 스토리는 정보 전달보다 **분위기 유지** 목적으로 사용합니다.

### 뉴스레터

- 제목은 질문형·감탄형 없이, 상태나 장면을 서술하는 형태로 씁니다.
- 이미지 1–2장 + 짧은 텍스트 구조를 기본으로 합니다.
- 발행 주기는 격주 또는 월 1회를 권장합니다.

### 상세페이지 (PDP)

- 도입부는 제품 스펙이 아니라 **감각 묘사**로 시작합니다.
- 소재·치수 등 정보는 후반부에 간결하게 정리합니다.
- 이미지 순서: 전체 컷 → 소재 클로즈업 → 공간 연출 컷 → 디테일

## 8. Do & Don't

| DO | DON'T |
|---|---|
| 여백을 디자인 요소로 씁니다 | 빈 공간을 채우려 하지 않습니다 |
| 물건이 먼저 말하게 합니다 | 브랜드가 앞에 나서지 않습니다 |
| 한 장면에 하나의 메시지 | 여러 정보를 한 화면에 몰지 않습니다 |
| 소재와 촉감을 보여줍니다 | 라이프스타일 이미지 남발은 지양합니다 |
| 절제된 컬러 사용 | 포인트 컬러를 여러 곳에 동시에 씁니다 |
